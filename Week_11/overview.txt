During this week we will begin our study of the Object Oriented design pattern: Inheritance
This will include the following concepts

    ~ Code reuse using inheritance
    ~ "has a" vs "is a" associations
    ~ Extending classes (super class and sub class)
    ~ Inheritance Hierarchies and classes of multiple types
    ~ Type casting objects in an inheritance hierarchy: implicit and explicit
    ~ Method over riding and the @Override annotation
    ~ These concepts form the functional basis for polymorphism
